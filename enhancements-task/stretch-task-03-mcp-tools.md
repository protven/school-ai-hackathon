# Задача со звездочкой 3. Подключить MCP tools

## Цель

Расширить возможности Open WebUI через MCP-инструменты.

## Что нужно сделать

- выбрать 1-2 простых MCP tool-сервера для демонстрации;
- подключить их к Open WebUI;
- убедиться, что инструменты доступны модели;
- выбрать только такие MCP-интеграции, которые дают понятную демонстрационную пользу.

## Ожидаемый результат

Модель может вызывать внешние инструменты через MCP и использовать их в ответах.

## Как проверить

- Open WebUI видит подключенные MCP tools;
- модель может вызвать инструмент в реальном сценарии;
- есть короткий demo-сценарий, показывающий пользу MCP.
