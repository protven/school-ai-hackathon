# Задачи со звездочкой

Это необязательные задачи после MVP.

Они не входят в минимальный прототип и не должны блокировать базовую реализацию. Их цель — показать, как проект можно развить после того, как уже работает локальный чат и простой RAG.

## В эту группу входят

1. Добавить reranker для улучшения качества retrieval
2. Добавить web search через SearXNG или внешний search API
3. Подключить MCP tools в Open WebUI
4. Написать собственные расширения на Python:
   - фильтр персональных данных
   - свой tool / function / plugin

## Принцип работы с этими задачами

- сначала должен быть готов MVP;
- каждая задача выполняется отдельно;
- задачи не должны ломать базовый сценарий;
- каждая задача проверяется отдельно.
