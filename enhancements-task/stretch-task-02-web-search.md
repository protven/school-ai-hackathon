# Задача со звездочкой 2. Добавить web search

## Цель

Дать модели возможность искать актуальную информацию в интернете и использовать ее в ответах.

## Что нужно сделать

- добавить web search в Open WebUI;
- в качестве первого варианта рассмотреть SearXNG;
- как альтернативу рассмотреть внешний search API через режим external;
- зафиксировать минимальную конфигурацию, которая не усложняет базовый локальный стенд.

## Ожидаемый результат

В системе появляется отдельный сценарий поиска информации в интернете, не связанный с локальными документами.

## Как проверить

- в Open WebUI включен web search;
- модель может выполнить поиск и использовать найденные результаты в ответе;
- базовый локальный RAG по документам продолжает работать независимо.
