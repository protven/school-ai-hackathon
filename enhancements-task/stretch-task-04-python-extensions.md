# Задача со звездочкой 4. Добавить собственные Python-расширения

## Цель

Показать, как проект можно расширять собственным кодом внутри Open WebUI.

## Что нужно сделать

Нужно подготовить хотя бы одно собственное расширение на Python.

Предпочтительные варианты:
- filter function для маскирования персональных данных;
- собственный tool / function / plugin для простой прикладной задачи.

## Ожидаемый результат

В проекте появляется собственное расширение, написанное под Open WebUI и решающее понятную локальную задачу.

## Как проверить

- расширение подключается в Open WebUI;
- его можно включить и использовать;
- есть короткий пример работы;
- расширение не ломает основной сценарий MVP.
