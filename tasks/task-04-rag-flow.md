# Задача 4. Подключить embedding-модель и проверить RAG

## Цель

Завершить минимальный локальный RAG-сценарий: загрузка документа, индексация и ответ по содержимому документа.

## Что нужно сделать

Нужно:

- настроить сервис `llama-embed` на запуск embedding-модели;
- подключить embedding endpoint в Open WebUI;
- проверить загрузку небольшого документа;
- проверить вопрос-ответ по содержимому документа.

На этом этапе нужно получить первый полностью рабочий RAG-сценарий.

## Ограничения

- Не добавлять OCR
- Не добавлять сложную обработку PDF
- Не выходить за рамки небольших demo-документов
- Не усложнять архитектуру дополнительными сервисами

## Ожидаемый результат

Работает минимальный локальный RAG-сценарий на обычном ноутбуке без GPU.

## Проверка результата

Результат считается достигнутым, если:

1. Документ загружается через Open WebUI
2. Документ индексируется
3. По документу можно задать вопрос
4. Система возвращает осмысленный ответ, связанный с содержимым документа
