# Задача 3. Поднять локальный чат через Open WebUI и llama.cpp

## Цель

Запустить минимальный локальный чат без RAG, чтобы сначала проверить базовую связку интерфейса и chat-модели.

## Что нужно сделать

Нужно:

- настроить сервис `llama-chat` на запуск chat-модели;
- подключить Open WebUI к локальному chat endpoint;
- убедиться, что обычный чат работает без использования RAG.

На этом этапе важно сначала доказать, что локальная LLM отвечает через интерфейс Open WebUI.

## Ограничения

- Не подключать embeddings раньше, чем заработает обычный чат
- Не усложнять конфигурацию лишними параметрами
- Не выходить за рамки минимального CPU-only сценария

## Ожидаемый результат

Open WebUI доступен в браузере и использует локальную chat model для ответов на обычные запросы.

## Проверка результата

Результат считается достигнутым, если:

1. `docker compose up -d` поднимает нужные сервисы
2. Open WebUI открывается локально
3. Из интерфейса можно отправить обычный запрос
4. Локальная chat model возвращает ответ
