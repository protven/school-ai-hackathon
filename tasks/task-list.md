# Задачи для агента-кодера

## Задача 1. Подготовить базовый каркас локального проекта

### Что нужно сделать
- Добавить `docker-compose.yml`
- Добавить `.env.example`
- Подготовить базовые каталоги `models/` и `data/`
- Зафиксировать имена сервисов:
  - `open-webui`
  - `llama-chat`
  - `llama-embed`

### Ожидаемый результат
В репозитории есть минимальный инфраструктурный каркас проекта для локального запуска.

### Как проверить
- В репозитории присутствуют нужные файлы и каталоги
- `docker compose config` выполняется без ошибок

---

## Задача 2. Подготовить загрузку моделей

### Что нужно сделать
- Добавить скрипт `download_models.sh`
- Скачивать chat-модель и embedding-модель в локальную папку `models/`
- Использовать зафиксированные модели:
  - `Qwen/Qwen3-1.7B-Instruct-GGUF`
  - `Qwen/Qwen3-Embedding-0.6B-GGUF`

### Ожидаемый результат
Модели скачиваются локально в понятные пути и готовы к использованию контейнерами.

### Как проверить
- Скрипт выполняется без ошибок
- После выполнения обе модели лежат в `models/`

---

## Задача 3. Поднять локальный чат через Open WebUI и llama.cpp

### Что нужно сделать
- Настроить `llama-chat` на запуск chat-модели
- Подключить Open WebUI к локальному chat endpoint
- Проверить, что обычный чат работает без RAG

### Ожидаемый результат
Open WebUI открывается в браузере, а локальная chat model отвечает на обычные вопросы.

### Как проверить
- `docker compose up -d` поднимает сервисы
- Open WebUI доступен локально
- Из интерфейса можно отправить простой запрос и получить ответ от chat model

---

## Задача 4. Подключить embedding-модель и проверить RAG

### Что нужно сделать
- Настроить `llama-embed` на запуск embedding-модели
- Подключить embedding endpoint в Open WebUI
- Проверить загрузку небольшого документа
- Проверить вопрос-ответ по содержимому документа

### Ожидаемый результат
Работает минимальный локальный RAG-сценарий: загрузка документа, индексация, поиск релевантных фрагментов и ответ по документу.

### Как проверить
- Документ загружается через Open WebUI
- Документ индексируется
- По документу можно задать вопрос
- Система возвращает осмысленный ответ, связанный с содержимым документа
