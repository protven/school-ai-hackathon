# Local RAG Demo

Это учебный проект для школьного хакатона.

Мы хотим собрать простой локальный AI-сервис, который умеет отвечать на вопросы по загруженным документам. Пользователь загружает файлы, пишет вопрос, а система находит нужные куски текста и формирует ответ.

Все должно работать локально на обычном ноутбуке, без сложной инфраструктуры и без GPU.

Стек:
- Open WebUI
- llama.cpp
- локальная LLM-модель
- embedding-модель
- Docker Compose
